# 🌙 Blue-Hour Study Space
A cozy digital study space that creates a calm and focused environment based on people's mood, with Pomodoro timer and relaxing musics. 

## Features:
- Pomodoro timer
- Mood selector with quotes
- Relaxing background musics
- Dynamic background colours based on mood
  
## Built with:
- HTML5
- CSS3
- JavaScript (Vanilla)
